# fortune-chess
「中華象棋神卦」是一種結合中國傳統象棋與易經卦理的占卜法，核心在於利用 32 枚棋子的文字象徵與五行關係來推算運勢。   "Chinese Chess Oracle" is a method of divination that combines traditional Chinese chess with the principles of the I Ching. Its core function is to use the symbolic characters and the Five Elements relationships inherent in the 32 chess pieces to predict fortune and outcomes.
